
En:
Test Task (Head Beating Mechanic):
Create a mechanic for head beating. The scene must include a head that can be hit. The damage dealt should be clearly visible.

Main Requirements:

Immersion

Juiciness (impactful and satisfying feel)

Creativity

Additional Requirements:

Must be implemented in 3D

Preferably in VR (if no headset is available, non-VR is acceptable)

Publish the project publicly on GitHub for review

Include a gameplay recording with commentary (voice or text)

Use Unity version 2022.3.xx (LTS) — Unity archive link

Only use copyright-free assets




Ру:
Тестовое задание (битье головы):
Сделать механику избиения головы. На сцене должна присутствовать голова с возможностью ее бить. Нанесение урона должно быть заметно.
Главные требования: иммерсивность, смачность, креатив.
Дополнительные требования:
Реализация должна быть в 3D.
Желательно в VR (а если шлема нет нет то тогда можно не в VR).
Опубликовать проект в общий доступ на GitHub для проверки.
Прикрепить запись геймплея с комментарием обзора (голосом/текстом).
Версия Unity 2022.3.xx (LTS) unity archive link.
Использовать только copyright free ассеты.
